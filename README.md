# Client_analysis

## Project overview
This project aims to build a complete data analysis pipeline using marketing data.

## Technical stack 
  - Raw dataset from Kaggle: https://www.kaggle.com/datasets/imakash3011/customer-personality-analysis
  - ETL: KNIME (data cleaning and normalization)
  - Power Query: Transformation
  - PowerBI: Creating DAX metrics and visualizing *

## Data pipeline 
Raw data -> KNIME ETL -> Clean dataset -> PowerBI -> Dashboard

## Dashboard
  - Page 1: Demographics (KPI, age distribution, education breakdown, marital status, income distribution, age x income matrix)
  - Page 2: Purchasing Behavior (spendind by product category, purchase channels, scatter: income vs total spent)
  - Page 3: Campaign performance (KPI, campaign response rates, customer response based on education/ marital status/ income and age group)
  - Page 4: Segmentation (Client profile distribution)

## Insight
  - Mid-income customers tend to spend more money that others incomes groups, especially on wine products. 
  - 2nd marjeting campaign was i huge flop regarding others
  - Customer segmentation reveals high-value and low-value groups
